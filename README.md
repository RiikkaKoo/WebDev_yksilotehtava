### Sofware Development: Web Development Course

## Restaurant API personal assignment

This is the repository for the personal assigment for Web Development course. <br>
The assignment is to make a web app that lists student restarants and shows their daily and weekly menus. <br>
Restaurants are shown on a map. <br>
Users can create profiles and login to an already existing profile. <br>
Users can change their information (username, email and password) as well as their avatar. <br>

The app is available through this link: https://users.metropolia.fi/~riikaka/OHTU_WebDevelopmentTasks/Yksilotehtava/
